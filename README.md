# cross_compiler_project

C++ Project to test linkage with GCC 7.3.1 binaries against pre C++11 GCC 3.4.6 binaries.

# Build library with gcc 3.4.6

 - cd gcc3.4.6
 - bash build.sh
 - bash install.sh 
 
# Build executable under gcc 7.3.1

 - cd gcc7.3.1
 - bash build.sh
 
# Run

 - ./bin_unix/cross_compiler_project
